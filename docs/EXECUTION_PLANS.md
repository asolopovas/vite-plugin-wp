# Execution Plans

Use a lightweight chat plan for small work. Check in a plan under `docs/exec-plans/active/` for complex, risky, or multi-session work; move it to `docs/exec-plans/completed/` when done.

## Template

```md
# <Task>

## Goal

## Scope

## Acceptance criteria

## Progress log

## Decisions

## Validation

## Follow-up debt
```

Keep plans current while working. Record validation commands and outcomes before handoff.
